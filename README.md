# E-Coder - Web Development Platform

A modern, responsive web development project built with **HTML5, CSS3, Bootstrap 5, and JavaScript**.

## Live Demo

[View Live Site](https://malik23m.github.io/E-coder/) *(After GitHub Pages deployment)*

## Features

-  Responsive design for all devices
-  Modern UI with Bootstrap 5
-  Interactive components with JavaScript
-  Clean and organized code structure
-  Cross-browser compatibility
-  Fast loading optimized images

##  Pages

- **Home** - Main landing page
- **About** - About the platform
- **Courses** - Available courses
- **Contact** - Contact form
- **Login** - User login page
- **Team** - Team members
- **Testimonials** - User reviews

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling
- **Bootstrap 5** - Responsive framework
- **JavaScript** - Interactivity
- **SCSS** - Enhanced CSS
